Boas Práticas

Semântica

O HTML semântico é a forma de deixar o site com suas informações bem explicadas e compreensíveis para o computador.

Layouts

Tabelas não deve ser usadas para layouts de páginas. O mais apropriado é seccionar os elementos de uma forma que eles envelopem a navegação principal (tag nav), rodapé (tag footer), unidades de conteúdo repetidas(tag article), etc.

CSS Reset

Resets corrige as inconsistências entre browsers.

IDs e Classes

IDs definem elementos únicos na sua página, enquanto classes são usadas para elementos que se repetem.
